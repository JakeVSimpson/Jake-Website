ISS Tracker

How it works?

The program fetches the current position of the ISS from the http://api.open-notify.org/iss-now.json endpoint and determines whether it's within 5-degrees latitude and 5-degrees longitude of a specified location. If the ISS is nearby the user will receive an email. In addition, the program utilizes the https://api.sunrise-sunset.org/json API to determine the times for sunrise and sunset at the user's location.
